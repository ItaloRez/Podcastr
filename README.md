Podcastr, Desenvolvimento da frontend de um site de podcast, acompanhando o evento NLW 5
 
 Como utilizar:
 
 * npm install
 
 Para rodar em modo de desenvolvimento:
 
 * yarn dev
 
 Para rodar em modo de produção:
 
 * yarn build
 * yarn start

* "axios": "^0.21.1", //substituir o fetch
